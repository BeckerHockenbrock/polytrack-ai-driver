## Purpose

The purpose of this project is **educational**. It is a personal exploration of computer vision, machine learning, imitation learning, and reinforcement learning, using a publicly playable game as a learning environment.

## Scope

This project:

- Does **not** modify the PolyTrack game or its files in any way.
- Does **not** read or write to PolyTrack's memory or process state.
- Does **not** interact with PolyTrack's network traffic, servers, or backend.
- Does **not** submit scores, ghosts, replays, or any other data to PolyTrack's leaderboards or online services.
- Does **not** interfere with other players or with the official game experience.

The agent perceives the game purely through screen capture and acts on the game purely through emulated keyboard input — the same channels available to any human player.
